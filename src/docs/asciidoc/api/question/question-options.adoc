===== Notes

* 이 API는 질문의 옵션과 투표 결과를 반환합니다.
* 질문자와 투표자는 항상 결과를 볼 수 있습니다.
* 마감되지 않은 질문에 대해 투표하지 않은 사용자는 결과를 볼 수 없습니다.
* 마감된 질문에 투표하지 않은 사용자는 결과를 볼 수 있습니다.

* `QUESTION_NOT_FOUND`: 존재하지 않는 질문을 요청할 때

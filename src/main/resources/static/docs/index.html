2. API

2.1. 인증

2.1.1. 카카오 로그인

HTTP Request
POST /api/v1/auth/kakao-login HTTP/1.1
Content-Type: application/json;charset=UTF-8
Content-Length: 100
Host: localhost:8080

{
  "authorizationCode" : "kakao authorization code",
  "redirectionUri" : "http://localhost:3000"
}
Notes

  • 이 API는 사용자가 카카오 계정을 통해 로그인할 때 사용됩니다.

  • 요청 바디에는 카카오로부터 받은 인가 코드가 포함되어 있어야 합니다.

HTTP Response
HTTP/1.1 200 OK
Content-Type: application/json;charset=UTF-8
Content-Length: 166

{
  "code" : 200,
  "message" : "OK",
  "data" : {
    "userId" : 1,
    "email" : "user@email.com",
    "token" : "token",
    "refreshToken" : "refresh token"
  }
}
Notes

  • 응답에는 사용자 ID, 이메일, 토큰 및 리프레시 토큰이 포함됩니다.

  • 유효하지 않은 인가 코드로 로그인을 시도하면 오류가 발생합니다.

Potential Errors

  • INVALID_TOKEN: 미인증 사용자가 요청할 때
